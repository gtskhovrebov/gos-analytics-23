# GosCore Analytics 23 v2.0

Created by John Lincoln.

## Что изменено
- компактная авторская карточка;
- дорогой dashboard с executive summary;
- новый рейтинг организаций;
- новая тепловая карта;
- причины индекса по факторам;
- сравнение недель с карточками;
- AI-сводка по категориям;
- полностью переработанный PNG executive report.

## Синхронизация
Используется закрытый лист Google Sheets `Stats` через Apps Script bridge.

Формат листа:
Организация | Показатель | ПН | ВТ | СР | ЧТ | ПТ | СБ | ВС
